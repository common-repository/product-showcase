== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `product-showcase.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('product_showcase_hook'); ?>` in your templates
